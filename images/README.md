# Project Visualizations
This folder contains the visualizations generated for the Sales & Demand Forecasting project.

## Included Visualizations
### 1. Monthly Sales Trend
Displays historical monthly sales patterns over time, helping identify trends and seasonality.
### 2. Sales by Category
Shows the total sales generated by each product category and highlights the highest-performing category.
### 3. Sales by Region
Illustrates sales distribution across different regions, helping identify the strongest markets.
### 4. Actual vs Predicted Sales
Compares actual sales values with model predictions to evaluate forecasting performance.
### 5. Future Sales Forecast
Displays forecasted sales for upcoming months based on historical sales patterns and the trained machine learning model.
## Purpose

These visualizations help stakeholders:
* Understand historical sales trends
* Identify seasonal patterns
* Analyze category and regional performance
* Evaluate forecasting model accuracy
* Support inventory, staffing, and business planning decisions

All visualizations were created using Python, Pandas, Matplotlib, and Scikit-learn.
